<h1 align="center">Lista de Tarefas</h1>
<h2>Descrição</h2>
<p>
  Projeto de uma lista de tarefas feito com Java Spring Boot, Thymeleaf, MySQL e o MaterializeCSS
</p>

<h2>Funcionalidades</h2>
<p>
  <ul>
    <li>
      Criar novas Tarefas
    </li>
    <li>
      Editar o título e status das tarefas já existentes
    </li>
    <li>
      Excluir tarefas
    </li>
  </ul>
</p>
